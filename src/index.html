04

Solution.

Pour réaliser cela, je vais m'orienter vers l'utilisation d'un logiciel préexistant, tel que CGRU, qui est une solution open source. L'objectif consistera donc à déposer le fichier source sur la plateforme web. Ensuite, le système le dirigera vers la ferme de rendu. Une fois le rendu accompli, l'étudiant recevra une notification ou un mail pour l'informer de l'achèvement du processus.
